Model constructor function. Any new instance is created by passing in a schema to define the structure of the data in each model.
Parameters:
Name Type Description
schema object An outline of the fields and pieces of data that are required by each data model. This may vary in each Model instance, e.g., the Players data model may have different required fields in its schema than the Teams data model.
